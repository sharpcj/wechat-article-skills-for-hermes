# 自动推荐规则

当用户未指定配图风格时，根据文章内容自动推荐 Type 和 Style。

## 按文章调性

| 调性 | 推荐风格 |
|------|---------|
| 正式、专业 | 扁平矢量、蓝图、商务 |
| 轻松、幽默 | 手绘、温暖 |
| 犀利 | 海报、极简 |

## 按文章内容信号

| 内容信号 | 推荐 Type | 推荐 Style |
|---------|-----------|-----------|
| 抽象概念、原理、隐喻 | concept | 扁平矢量、简约线条、手绘 |
| 教程、步骤、操作指南 | process | 扁平矢量、简约线条 |
| 知识框架、原理讲解、课程 | whiteboard | 手绘、简约线条 |
| 数据、统计、趋势、报告 | data-viz | 扁平矢量、商务 |
| 对比、评测、方案选择 | comparison | 扁平矢量、商务 |
| 架构、系统、模型、体系 | architecture | 蓝图、扁平矢量 |
| 知识全景、关系梳理、概览 | mindmap | 扁平矢量、简约线条 |
| 历程、里程碑、路线图 | timeline | 简约线条、商务 |
| Tips、要点列表、注意事项 | checklist | 扁平矢量、温暖 |
| 金句、核心观点、名言 | quote-card | 温暖、海报、极简 |
| 使用场景、产品展示、代入感 | scene | 温暖、水彩、手绘 |
| 故事、情感、散文、视觉呼吸 | atmosphere | 温暖、水彩、海报 |

## 配图密度

| 密度 | 规则 |
|------|------|
| 每节一图 | 每个 H2 小标题后一张 |
| 少图 | 仅封面 + 1-2 张关键配图 |
| 多图 | 每 2-3 段一张 |

默认按需配图：每个 H2 章节最多 1 张，宁缺毋滥。
