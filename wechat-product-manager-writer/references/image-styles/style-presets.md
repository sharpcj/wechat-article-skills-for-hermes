# 正文配图风格预设（产品经理视角）

按文章类型推荐 Type + Style 组合。

## 按文章类型

### AI 产品拆解

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `产品拆解` | architecture | 蓝图 | 产品架构、功能模块、设计逻辑 |
| `场景对比` | comparison | 扁平矢量 | 产品对比、方案选择、优劣分析 |
| `体验分析` | scene | 温暖 | 用户场景还原、使用体验展示 |

### 场景解决方案

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `方案架构` | architecture | 蓝图 | 技术方案架构、系统设计 |
| `实施流程` | process | 简约线条 | 搭建步骤、操作流程、工作流 |
| `效果展示` | comparison | 扁平矢量 | 方案前后对比、效果数据 |

### 效率提升实战

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `技巧清单` | checklist | 简约线条 | 技巧汇总、Tips 列表、要点梳理 |
| `工作流` | process | 扁平矢量 | 工作流优化、效率提升路径 |
| `效果对比` | comparison | 商务 | 前后效率对比、数据变化 |

### 产品方法论

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `方法论框架` | mindmap | 扁平矢量 | 方法论全景、知识框架 |
| `概念讲解` | concept | 扁平矢量 | 抽象概念具象化、隐喻 |
| `金句传播` | quote-card | 温暖 | 核心观点、方法论金句 |

### 行业观察

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `趋势时间线` | timeline | 简约线条 | 行业发展历程、趋势演变 |
| `格局对比` | comparison | 商务 | 行业格局、竞争对比 |
| `观点卡片` | quote-card | 海报 | 鲜明观点、评论金句 |

## 文章内容 → 预设推荐

| 文章内容信号 | 推荐预设 | 备选 |
|-------------|---------|------|
| 产品功能、架构、设计 | `产品拆解` | `方案架构` |
| 使用场景、体验、感受 | `体验分析` | `场景对比` |
| 方案搭建、实施、部署 | `方案架构` | `实施流程` |
| 步骤教程、操作指南 | `实施流程` | `技巧清单` |
| 技巧汇总、效率提升 | `技巧清单` | `工作流` |
| 前后对比、效果数据 | `效果对比` | `效果展示` |
| 方法论、框架、体系 | `方法论框架` | `概念讲解` |
| 核心观点、金句 | `金句传播` | `观点卡片` |
| 行业趋势、格局分析 | `格局对比` | `趋势时间线` |
| 发展历程、时间线 | `趋势时间线` | `格局对比` |
