# 正文配图风格预设

按文章类型推荐 Type + Style 组合。

## 按文章类型

### 科技与技术

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `科技解析` | concept | 蓝图 | 技术原理、AI 概念、抽象概念具象化 |
| `系统设计` | architecture | 蓝图 | 架构图、系统设计、技术栈 |
| `工具评测` | comparison | 扁平矢量 | 工具对比、框架评测、方案PK |

### 知识与教程

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `知识科普` | concept | 扁平矢量 | 概念解释、入门指南、隐喻科普 |
| `操作指南` | process | 简约线条 | 分步教程、操作流程、工作流 |
| `效率工具` | checklist | 简约线条 | 产品指南、SaaS 介绍、Tips 列表 |
| `课堂笔记` | whiteboard | 手绘 | 知识框架、原理讲解、课程笔记 |
| `知识全景` | mindmap | 扁平矢量 | 知识框架、主题概览、关系梳理 |

### 数据与分析

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `数据报告` | data-viz | 商务 | 数据新闻、指标报告、统计分析 |
| `趋势对比` | comparison | 扁平矢量 | 行业对比、趋势分析 |
| `发展历程` | timeline | 简约线条 | 里程碑、产品路线图、年度总结 |

### 故事与情感

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `故事叙述` | scene | 温暖 | 个人随笔、成长故事、用户场景 |
| `生活方式` | atmosphere | 水彩 | 旅行、生活、文艺、散文 |
| `金句传播` | quote-card | 温暖 | 核心观点、名言金句、截图转发 |

### 观点与评论

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `观点文章` | atmosphere | 海报 | 社论、评论、犀利观点 |

## 文章内容 → 预设推荐

根据文章内容信号自动推荐：

| 文章内容信号 | 推荐预设 | 备选 |
|-------------|---------|------|
| API、代码、技术原理 | `科技解析` | `系统设计` |
| 概念、科普、入门指南 | `知识科普` | `课堂笔记` |
| 教程、步骤、操作 | `操作指南` | `知识科普` |
| 对比、评测、选择 | `工具评测` | `趋势对比` |
| 数据、报告、指标 | `数据报告` | `趋势对比` |
| 知识框架、关系梳理 | `知识全景` | `课堂笔记` |
| 历程、里程碑、路线 | `发展历程` | `数据报告` |
| Tips、要点、注意事项 | `效率工具` | `知识科普` |
| 故事、经历、场景 | `故事叙述` | `生活方式` |
| 金句、核心观点 | `金句传播` | `观点文章` |
| 观点、评论、争议 | `观点文章` | — |
| 工具、产品、SaaS | `效率工具` | `知识科普` |
